# 🏥 Health Dataset Preprocessing Pipeline

## 📌 Overview
This project performs data preprocessing on a health dataset to prepare it for machine learning.

## 🚀 Features
- Data cleaning  
- Missing value handling  
- Duplicate removal  
- One-hot encoding  
- Correlation analysis  

## 🛠️ Tech Stack
- Python  
- Pandas  
- NumPy  
- Scikit-learn  



## 📊 Output
- clean_health_dataset.csv  
- encoded_health_dataset.csv
